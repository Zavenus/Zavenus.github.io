Circle & Rectangle — Overlap demo

Two shapes (a circle and a rectangle) with different colors and adjustable opacity to show the overlap.

To upload: save this file as index.html and host it (GitHub Pages / Netlify / Vercel). See instructions below.
Quick usage
  1. Open the file locally (double-click index.html) to preview in your browser.
  2. Change colors/opacities with the controls to see the overlap effect.
Upload instructions (GitHub Pages, Netlify, Vercel)
GitHub Pages (recommended simple): 
git init
git add index.html
git commit -m "Add circle-rectangle demo"
# create a repo on github.com (e.g. circle-rect-demo)
# replace  with the repo url
git remote add origin 
git branch -M main
git push -u origin main
# then on GitHub: Settings → Pages → choose branch 'main' / folder '/' → Save
# URL will be https://<your-username>.github.io/<repo-name>/
Netlify (drag & drop): 
# Zip your index.html (or place in folder)
# Go to app.netlify.com/site/drop and drag the folder or index.html
# Netlify will give you a live URL instantly.
Vercel (quick CLI): 
npm i -g vercel
vercel login
vercel --prod
# follow prompts and pick the folder containing index.html
# you'll receive a public URL from Vercel

After you have the public URL, paste it into your portal where it asks for the link.
